# Installation

## Method 1: Installation from Conda (OSX and Linux only)

This package has been added to the 54gene [Conda](https://docs.conda.io/en/latest/) channel.
To install, first install and configure [miniconda](https://docs.conda.io/en/latest/miniconda.html).

From the command line, execute the following command:

```
conda install mamba
mamba install -c conda-forge -c "https://gitlab.com/data-analysis5/conda-54gene/-/raw/default/conda-54gene" r-process.phenotypes
```

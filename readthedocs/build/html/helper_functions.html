Helper FunctionsÔÉÅ

This package includes a variety of exposed helper functions that can be called from the configuration files, which are generally used either in cross-variable dependencies or creation of derived variables. All of these helper functions can be elaborated in the usual way in an R terminal (e.g. ?process.phenotypes::derive.rank.normal.transform), but this page provides a brief listing of the functions available.
